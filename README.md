# Machine Learning Foundations — Week 1 Assignment

This repository contains the completed **Week 1 Assignment** covering fundamental math, statistics, and programming building blocks required for Machine Learning and Data Science.

## 🚀 Topics Covered
* **Python Fundamentals:** Control flow, custom exceptions, data structures, and lambda functions.
* **NumPy:** Multi-dimensional array manipulation, boolean indexing, slicing, and matrix operations.
* **Pandas:** Data loading, filtering, structural aggregation (`groupby`), and handling missing datasets.
* **Linear Algebra:** Matrix multiplications, $L_2$ norm visualizations, Eigenvalues/Eigenvectors, and Singular Value Decomposition (SVD).
* **Statistics & Probability:** Descriptive stats, Hypothesis testing (t-test, Pearson correlation), error metrics from scratch (MAE, MSE, $R^2$), Central Limit Theorem (CLT), and Bayes' Theorem.

## 📂 Repository Layout
* `week1_Sameer.ipynb`: Core notebook containing code solutions and theoretical reflections.
* `requirements.txt`: Python packages required to execute the notebook.
